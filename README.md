# R&D репозиторий по дизайну A/B-экспериментов

Репозиторий реорганизован в формат **4 независимых RnD-тем**. Для каждой темы предусмотрены отчёт и ноутбук с воспроизводимыми расчётами.

## Структура

- `rnd_01_статьи_и_математика/`
  - `report.md` — теоретический обзор (статьи, формулы, математика).
  - `analysis.ipynb` — короткие расчёты по росту вероятности ложных находок.
- `rnd_02_множественные_проверки/`
  - `report.md` — рекомендации по коррекциям в финальном анализе.
  - `analysis.ipynb` — демо Bonferroni и Benjamini–Hochberg.
- `rnd_03_рерандомизация/`
  - `report.md` / `report.pdf` — подробный отчёт по балансировке и рерандомизации.
  - `analysis.ipynb` — компактное демо по риску ложного сигнала.
  - `simulation.py` — воспроизводимая симуляция правил приемки сплита.
  - `simulation_results.csv` — пример результата симуляций.
- `rnd_04_план_внедрения/`
  - `report.md` — операционный план внедрения.
  - `analysis.ipynb` — чек-лист подготовки эксперимента.
- `tools/`
  - `execute_notebooks.py` — выполняет все ноутбуки и сохраняет output.
  - `generate_pdf.py` — пересобирает `rnd_03_рерандомизация/report.pdf`.
- `requirements.txt` — базовые зависимости для работы с Jupyter.
- `AGENTS.md` — зафиксированные правила ведения этого репозитория для последующих итераций.

## Как воспроизводить

### 1) Установить зависимости
```bash
pip install -r requirements.txt
```

### 2) Выполнить все ноутбуки
```bash
python tools/execute_notebooks.py
```

### 3) Запустить симуляцию рерандомизации
```bash
python rnd_03_рерандомизация/simulation.py --n-iter 500 --n 500 --feature-grid 5,10,20,50 --output rnd_03_рерандомизация/simulation_results.csv
```

### 4) Пересобрать PDF-отчёт
```bash
python tools/generate_pdf.py
```

## Принципы оформления
- Отчёты и комментарии в коде — на русском языке.
- В отчётах есть ссылки на соответствующие элементы кода.
- Ноутбуки коммитятся только в исполненном состоянии (с обновлёнными output).
