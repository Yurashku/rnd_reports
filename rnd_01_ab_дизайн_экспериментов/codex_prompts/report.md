# Краткий отчёт: библиотека лучших промптов для Codex

## Цель
Собрать единый практический набор промптов, который ускоряет типовые инженерные задачи: анализ, внесение изменений, дебаг, рефакторинг и подготовку PR.

## Что добавлено
- Каталог `codex_prompts/` внутри реализованного RnD.
- Файл `prompts_codex.md` с 7 шаблонами высокого качества.

## Как использовать
1. Выбрать шаблон под сценарий.
2. Подставить контекст проекта (цель, ограничения, артефакты).
3. Зафиксировать проверки и ожидаемый формат ответа.
