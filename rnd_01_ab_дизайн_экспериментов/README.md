# RnD 01: дизайн A/B-экспериментов

## Главная точка входа

Открывайте один файл:
- **`report.md`** — консолидированный отчёт по всему RnD (цели, формулы, решения, риски, план внедрения).

## Что лежит рядом
- Подпапки `01_* ... 04_*` содержат ноутбуки и детальные рабочие материалы.
- Основной воспроизводимый скрипт симуляции: `03_рерандомизация/simulation.py`.
- Если нужен расширенный текстовый вариант по рерандомизации: `03_рерандомизация/report.md` и `report.pdf`.

Идея структуры: содержательные выводы читаются в одном файле, а код открывается только при необходимости проверки воспроизводимости.
