# AGENTS instructions for this repository

## Обязательные правила работы
1. При любом изменении в репозитории обновляй `README.md`, чтобы он оставался понятным и актуальным.
2. Все ноутбуки `*.ipynb` перед коммитом должны быть полностью исполнены и сохранены с актуальными output.
3. Если в репозитории есть tutorial-материалы, их нужно обновлять при изменениях процесса/команд.
4. Отчёты и комментарии в коде — на русском языке.
5. Для каждого RnD важно хранить:
   - краткий отчёт,
   - воспроизводимый и легко читаемый код симуляций (предпочтительно в Jupyter),
   - ссылки из отчёта на соответствующий код.
