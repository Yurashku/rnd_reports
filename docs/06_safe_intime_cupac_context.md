# Контекст R&D-6: Safe in-time covariates для CUPAC

- **Версия документа:** v0.1
- **Статус:** Draft (заглушка)
- **Связанный RnD:** `06_safe_intime_cupac/`

## Зачем
Снижение дисперсии метрик (CUPED/CUPAC) ускоряет эксперименты, но использование
произвольных ковариат рискованно: ковариата, зависящая от назначения или
собранная «слишком поздно», может внести смещение. Трек R&D-6 фокусируется на
**безопасных in-time ковариатах** — признаках, доступных по ходу теста и не
нарушающих валидность оценки эффекта.

## Что входит в трек
- Формализация понятия «безопасная ковариата» — см. [политику](feature_safety_policy.md).
- Методология снижения дисперсии — см. [методологию](variance_reduction_methodology.md).
- Общий код в `src/rnd_reports/` (`variance_reduction`, `feature_safety`, `synthetic`, `benchmark`).
- Конфиги сценариев/наборов/методов в `configs/06_safe_intime_cupac/`.

## Принципы
- Эксперименты — только на синтетике, реальные данные не коммитятся.
- Library-first: CUPAC берётся из HypEx, при отсутствии — блок помечается недоступным.
- На текущем этапе кода-алгоритмов нет, только структура и заглушки.
