# Будущие RnD (ещё не реализованы)

Эта папка зарезервирована под новые RnD-направления.

## Принцип именования
- `rnd_02_<название>`
- `rnd_03_<название>`
- и т.д.

## Минимальный состав каждого нового RnD
1. `report.md` — краткий отчёт с выводами.
2. `analysis.ipynb` — воспроизводимый ноутбук с актуальными output.
3. (Опционально) скрипты симуляций и артефакты (`*.csv`, `*.pdf`).
4. Ссылки из отчёта на соответствующий код.
