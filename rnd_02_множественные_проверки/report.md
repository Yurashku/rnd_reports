# RnD 02: множественные проверки в финальном анализе

## Вопрос
Нужно ли корректировать p-value по нескольким продуктовым метрикам в финальном A/B-анализе?

## Короткий ответ
Да, если решение принимается по семейству гипотез, иначе растёт риск ложноположительных выводов.

## Практика
- Для строгого контроля: Bonferroni/Holm.
- Для баланса мощности и контроля ошибок: Benjamini–Hochberg (FDR).
